MEDIDAS

O QUE É MEDIDA?


Medida é o processo de comparar uma grandeza com um padrão para saber “quanto” daquela coisa existe.

Explicando de forma simples
Quando você mede algo, está respondendo perguntas como:
“Qual é o tamanho?”
“Qual é o peso?”
“Qual é a temperatura?”
Por exemplo:
Dizer que uma mesa tem 2 metros → você comparou com o padrão “metro”
Dizer que está 30°C → comparou com a escala de temperatura

NA CIENCIA
Em áreas como a Física, medir é essencial para descrever o mundo com precisão.
Uma medida sempre tem:
Número (valor)
Unidade (metro, kg, segundo, grau, etc.)
Exemplo: 10 kg
10 = número
kg = unidade

TIPOS DE MEDIDAS
Algumas das principais:
Comprimento (metro)
Massa (quilograma)
Tempo (segundo)
Temperatura (grau Celsius)

RESUMINDO Medida = número + unidade que representa uma quantidade comparada a um padrão.

O QUE É PRECISÃO DE MEDIDA?

Precisão de medida é o quanto uma medição é consistente e próxima de outras medições feitas do mesmo jeito.


Explicando fácil
Se você mede a mesma coisa várias vezes e os resultados dão quase iguais, então a medida é precisa.
Exemplo:
Medidas: 10,1 cm / 10,2 cm / 10,1 cm
Alta precisão (valores muito próximos)

CUIDADO: precisão ≠ exatidão
Na Física, esses termos são diferentes:
Precisão → valores próximos entre si
Exatidão → valor próximo do real
Exemplo:
9,9 / 9,8 / 9,9 (se o real é 10,0)
Preciso = correto Não exato = errado

O QUE AFETA A PRECISÃO
Qualidade do instrumento
Forma de medir
Erros humanos
Ambiente (temperatura, vibração, etc.)

O QUE É EXATIDÃO?

Exatidão é o quanto uma medida está próxima do valor real (verdadeiro).


EXPLICANDO
Se você mede algo e o resultado fica muito perto do valor correto, então a medida é exata.
Exemplo:
Valor real: 10,0 cm
Medida: 10,0 cm
Alta exatidão

DIFERENÇA IMPORTANTE
Na Física:
Exatidão → perto do valor real
Precisão → medidas parecidas entre si
Exemplo:
10,0 cm / 10,1 cm / 9,9 cm
Exato (perto do real)
Preciso também
Outro:
8,0 / 8,1 / 8,0
Preciso = correto
Não exato = errado (longe do real)

RESUMINDO Exatidão = acerto (proximidade do valor verdadeiro)
Quanto mais perto do valor real, mais exata é a medida

O QUE É A PONTE DE WHEATSTONE

A Ponte de Wheatstone é formada por quatro resistores organizados em forma de ponte (um losango),
com uma fonte de tensão aplicada e um medidor no meio.

ESTRUTURA BÁSICA
Ela tem:
4 resistores (R1, R2, R3 e Rx)
1 fonte de tensão
1 detector (galvanômetro ou voltímetro)
O resistor Rx é o valor desconhecido que queremos medir.

COMO FUNCIONA
A ponte funciona com base no equilíbrio de tensão.
Quando está balanceada:
Não passa corrente pelo detector
A relação entre os resistores é:
R1/R2 = R3/Rx
Daí você calcula o valor de Rx.

APLICAÇOES PRÁTICAS
A Ponte de Wheatstone é muito usada com sensores que variam resistência, como:
Strain gauges (medição de deformação)
Sensores de pressão
Sensores de temperatura (RTD)
Células de carga (balanças)

EXEMPLO SIMPLES
Numa balança eletrônica, quando você coloca peso:
A resistência muda
A ponte fica “desequilibrada”
Surge uma tensão de saída proporcional ao peso

O QUE É UM MULTIMETRO

Um multímetro é um instrumento usado para medir grandezas elétricas em circuitos,
como tensão, corrente e resistência. Ele é essencial tanto para eletrônica básica quanto para manutenção elétrica.

multímetro digital é um aparelho que reúne várias funções de medição em um só equipamento.

PRINCIPAIS MEDIÇOES
Tensão (Voltagem)
Mede a diferença de potencial
Pode ser:
DC (corrente contínua) → pilhas, baterias
AC (corrente alternada) → tomadas

CORRENTE (Amperagem)
Mede o fluxo de elétrons no circuito
Unidade: Ampère (A)
Normalmente precisa ligar o multímetro em série no circuito

RESISTENCIA
Mede a oposição à passagem de corrente
Unidade: Ohm (Ω)

TIPOS DE MULTIMETRO
📟 Digital
Mostra valores em tela
Mais preciso e fácil de usar
Mais comum hoje
🎚️ Analógico
Usa ponteiro
Menos preciso
Mais raro atualmente

O QUE É MEDIDA DE CONTINUIDADE

A medida de continuidade é um teste feito com um multímetro para verificar se existe
um caminho elétrico fechado entre dois pontos — ou seja, se a corrente consegue passar.

O QUE É CONTINUIDADE?
Quando há continuidade, significa que:
O circuito está fechado
Não há interrupções (fio rompido, trilha quebrada, etc.)
Quando não há continuidade:
O circuito está aberto
A corrente não passa

COMO FUNCIONA NO MULTIMETRO?
No modo de continuidade do multímetro digital:
O aparelho envia uma pequena corrente pelo circuito
Se a resistência for muito baixa:
Ele emite um bip (som)
Se for alta ou infinita:
Não apita

EXEMPLOS DE USO
Verificar se um fio está rompido
Testar fusíveis
Checar conexões elétricas
Identificar trilhas em placas eletrônicas

IMPORTANTE
Sempre testar com o circuito desligado
Não usar em circuitos energizados
