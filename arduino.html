ARDUINO


O QUE É O ARDUINO E QUAL SUA ESCALA ANALÓGICA E DIGITAL?


O Arduino é uma plataforma de prototipagem eletrônica de hardware livre e código aberto,
baseada em microcontroladores fáceis de programar.
Ele permite criar projetos interativos lendo sensores (analógicos/digitais) e controlando atuadores (luzes, motores).

ESCALA ANALÓGICA (ADC)
O Arduino converte sinais analógicos em números digitais.
Resolução: 10 bits
Faixa padrão: 0 a 1023
Tensão típica: 0 a 5V
Ou seja:
0 → 0V
1023 → 5V
Cada passo ≈ 4,88 mV (5V / 1023)

ESCALA DIGITAL
Entradas e saídas digitais só têm dois estados:
LOW (0) → 0V
HIGH (1) → 5V
Funciona como um interruptor:
Desligado (0)
Ligado (1)

PWM (analógico “simulado”)
Alguns pinos digitais fazem PWM (Pulse Width Modulation):
Escala: 0 a 255 (8 bits)
Simula saída analógica variando o tempo ligado/desligado
Exemplo:
0 → sempre desligado
255 → sempre ligado
127 → ~50% ligado

RESUMO RÁPIDO
Analógico (entrada): 0 a 1023
Digital: 0 ou 1
